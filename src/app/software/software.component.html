SymAware Framework

Overview

The SymAware framework is a modular, extensible collection of Python packages, with the goal of providing a unified software infrastructure for real-time control of autonomous agents (e.g., drones, vehicles) in complex, dynamic environments. It comes with out-of-the-box implementations of core components, such as situational-aware controllers, perception systems, and communication modules, and it can interface with different simulators (e.g., Pybullet, Prescan, Carla) and ROS-based real-world deployments.

Installation

Install the base package via pip:

pip install \
"symaware-base@git+https://gitlab.mpi-sws.org/sadegh/eicsymaware.git@base"

Include support for a specific simulator (e.g., Pybullet):

pip install \
"symaware-pybullet@git+https://gitlab.mpi-sws.org/sadegh/eicsymaware.git@pybullet"

Or install all supported simulator interfaces & ROS at once:

pip install \
"symaware-base[simulators,extra]@git+https://gitlab.mpi-sws.org/sadegh/eicsymaware.git@base"

Key Components

  • Controller Computes control inputs using the agent's awareness and knowledge databases.
  • PerceptionSystem Collects and integrates sensory information from the environment.
  • CommunicationSystem Sends and receives messages between agents (TCP, UDP, ROS, custom protocols).

Running the examples

The examples folder contains several use cases demonstrating the framework's capabilities. Some examples require specific simulators to be installed and configured, while others can be run in a simple Python environment.

To run an example, clone the repository, install the desired simulator interfaces (e.g., symaware-pybullet), and execute the script, e.g., 

python3 examples/drone_main.py
